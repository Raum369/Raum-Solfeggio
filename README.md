# 🌌 Raum Solfeggio — Sanctuary of Sound

**Raum Solfeggio** (Solfeggio Sanctuary) — це преміальний мінімалістичний веб-застосунок для глибоких медитацій, сну та концентрації. Застосунок генерує чисті математичні частоти Сольфеджіо та бінауральні ритми в реальному часі безпосередньо у вашому браузері, поєднуючи преміальний дизайн у стилі **Apple / Endel** та науково обґрунтований аудіо-синтез.

---

## ✨ Особливості дизайну та інтерфейсу (Aesthetics)

- **Apple Minimalist & Endel Style:** Глибокий оксамитово-чорний фон, плавна сакральна геометрія та орбітальні кола, що налаштовують на спокій.
- **Constellation Particles (Сузір'я):** Динамічна інтерактивна Canvas-анімація нічного неба зі зірками, що плавно з'єднуються сузір'ями при наближенні.
- **Fluid Morphing Bubble (Сфера дихання):** Центральний елемент інтерфейсу, що плавно деформується в ритмі спокійного дихання в режимі очікування та вібрує відповідно до частот та гармонік під час відтворення.
- **Zero UI (Режим глибокого занурення):** При запуску аудіо всі кнопки, описи та елементи керування плавно зникають (за 2.5 секунди), залишаючи вас наодинці з пульсуючою сферою та звуком. Звичайна пауза повертає інтерфейс назад.
- **Плавні переходи кольорів та інерція:** Будь-які зміни кольорів при перемиканні частот інтерполюються покадрово. Завдяки 64-канальному низькочастотному фільтру (Low-Pass Smoothing), сфера реагує на звук надзвичайно пластично, як рідина.

---

## 🎧 Аудіо-двигун (Advanced Web Audio Engine)

На відміну від багатьох сервісів, які програють готові стислі аудіофайли, **Raum Solfeggio** використовує потужний **Web Audio API** для генерації звуку на льоту:

- **True Fundamental Tuning:** Частоти не транспонуються з класичної сітки 440 Гц, а генеруються як чиста математична основа.
- **Багатошаровий Drone Pad:**
  - **Fundamental Layer** — основа звуку.
  - **4 гармоніки** (x2, x3, x4, x5) для створення теплого, багатого аналогового забарвлення.
  - **Sub-Bass Layer** (0.5x частоти) — додає глибокого оксамитового низу, який відчувається тілом.
- **Stereo Filtered Wind Noise:** Шар легкого білого шуму, пропущений через смуговий low-pass фільтр, який повільно модулюється LFO (0.08 Гц), імітуючи ритм дихання океану чи вітру.
- **Binaural Beats (Бінауральні ритми):** При активації хвиль лівий та правий аудіоканали отримують зсув на обрану частоту (Delta, Theta, Alpha, Beta, Gamma) для делікатної стимуляції хвиль головного мозку (рекомендовано слухати в навушниках).
- **Convolution Reverb:** Синтетично згенерований імпульсний відгук з експоненціальним затуханням створює ефект безмежного космічного простору (cathedral-like reverb) без завантаження великих файлів.
- **Плавний Crossfade:** Зміна частоти відбувається через 3-секундний кросфейд, без різких клацань чи зупинок.

## 🛠️ Функціональні можливості

- **Розумний таймер:** Сесії на 15 хв, 30 хв, 1 год або безкінечно. За 10 секунд до кінця сесії гучність плавно згасає до нуля.
- **Автозбереження стану (Persistence):** Додаток автоматично запам'ятовує останній вибір частоти, бінауральної хвилі, гучності та таймера.
- **Haptic Feedback:** Легкий тактильний відгук на мобільних пристроях при взаємодії з елементами інтерфейсу.
- **Cache-Busting:** Механізм версіонування скриптів і стилів для миттєвого відображення оновлень у клієнта.

## 🚀 Як запустити локально

Оскільки додаток є повністю статичним, його можна запустити за допомогою будь-якого локального веб-сервера.

### Спосіб 1: Через Python (найпростіший)
Відкрийте термінал або PowerShell у папці з проектом та виконайте:
```bash
python -m http.server 8080
```
Після цього відкрийте у браузері:
[http://localhost:8080](http://localhost:8080) або [http://127.0.0.1:8080](http://127.0.0.1:8080)

### Спосіб 2: Live Server у VS Code
Якщо ви користуєтеся VS Code, встановіть розширення **Live Server** та натисніть кнопку **"Go Live"** у правому нижньому кутку редактора.
